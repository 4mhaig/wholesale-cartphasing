# Wholesale Cart - Delivery Phasing Prototype

Interactive prototype for wholesale cart delivery phasing with RID-RED validation.

## 🎯 Demo Scenarios

### Scenario 1: Partial Available
Articles with RID-RED windows that allow phasing to some delivery dates but not others. Test auto-split to see how quantities distribute only across valid dates.

### Scenario 2: Fixed RDD
Articles with mandatory fixed delivery dates. Each article can only be allocated to its specific month. Try transferring between dates to see validation in action.

### Scenario 3: Available Later
Articles not yet available for any existing delivery dates. The system prompts you to create new compatible dates when needed.

## 🧪 Testing

Feel free to explore beyond these scenarios! You can add more articles, create additional delivery dates, or test any workflow you'd like.

**Note:** This is an early prototype, so some features may not work perfectly. Apologies in advance for any rough edges—your feedback will help us improve it! 🙏

## 🚀 Features

- ✅ RID-RED date validation
- ✅ Fixed RDD support
- ✅ Auto-split with smart distribution
- ✅ Transfer between compatible dates
- ✅ Real-time allocation tracking
- ✅ Filters by allocation status and RDD

## 📝 Usage

1. Select a demo scenario from the top bar
2. Navigate between "Articles & Quantities" and "Delivery Phasing" tabs
3. Use filters to focus on specific articles
4. Allocate quantities to delivery dates
5. Try auto-split, transfer, or manual allocation
